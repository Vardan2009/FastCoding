How to Play

Training


This game is about training your hands.
A random key on the keyboard will light up, click the key on your keyboard with the matching finger



Typing


This game is about writing words as fast as possible
Write the word shown on the screen as fast as you can!
Every correct word will add +5 seconds to the timer
Every wrong one, -5
If the timer runs out, the game ends



Listening



This game is very similar to the "typing" game
Write the word you hear
Same rules from the "Typing" game apply to this game too





Reaction



This game is a reaction game
Click the letters in the alphabetical order to win!




Memory



This game is a memory game
Click a box to reveal its letter. Click all the letters with its pairs to win!
